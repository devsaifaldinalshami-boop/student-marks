# نظام عرض علامات الطلاب

مشروع ويب بسيط باللغة العربية لإدارة علامات الطلاب وعرضها للأهالي مع طباعة تقرير A4 لكل طالب.

## طريقة التشغيل

افتح الملف `index.html` مباشرة في المتصفح، أو شغل خادمًا محليًا من داخل مجلد المشروع:

```powershell
python -m http.server 8000
```

ثم افتح:

```text
http://localhost:8000
```

## لوحة الإدارة

افتح `admin.html` أو اضغط رابط "لوحة الإدارة" من الصفحة الرئيسية.

كلمة المرور التجريبية:

```text
qobaa2026
```

الحماية هنا بسيطة ومناسبة للاستخدام المحلي فقط، لأن كلمة المرور موجودة داخل ملفات الواجهة. عند نشر النظام على الإنترنت يفضل تحويله إلى Flask أو Django مع تسجيل دخول حقيقي.

تحتوي لوحة الإدارة أيضًا على جدول لترتيب الطلاب من الأعلى علامة إلى الأقل، مع إمكانية عرض الترتيب لكل الحلقات أو لحلقة محددة، ويمكن طباعته أو حفظه PDF من زر "طباعة الترتيب PDF".

تم تقسيم لوحة الإدارة إلى ثلاث خانات: "صف سابع وثامن" للعلامات الموجودة سابقًا، و"صف سادس" للعلامات المستوردة من ملف الصف السادس، و"صف خامس" للعلامات المستوردة من ملف الصف الخامس. عند اختيار القسم تتغير قائمة الطلاب والترتيب والحلقات المعروضة داخل الإدارة.

ملاحظة: حلقة الصحابي سلمان الفارسي محسوبة من 70 فقط، ولا تظهر فيها مادة الإضاءات.

## إضافة الطلاب والعلامات

هناك طريقتان:

1. من لوحة الإدارة: أضف طالبًا أو عدل بياناته ثم اضغط "حفظ الطالب". تحفظ التغييرات داخل المتصفح باستخدام `localStorage`.
2. من ملف البيانات: افتح `data.js` وعدل مصفوفة `students`. الملف الحالي يحتوي على بيانات مستوردة من ملفات الإكسل، ويمكن تعديلها أو إضافة طلاب جدد. كل طالب يحتوي على:

```js
{
  id: "student-unique-id",
  name: "اسم الطالب",
  className: "الفئة أو الصف",
  section: "الحلقة أو الشعبة",
  gradeGroup: "صف سابع وثامن",
  notes: "ملاحظات اختيارية",
  subjects: [
    { name: "الحفظ", score: 95, max: 100 },
    { name: "التجويد", score: 90, max: 100 }
  ]
}
```

## البحث

البحث مرن ويدعم اللغة العربية. يمكن كتابة جزء من الاسم، ويتجاهل التشكيل، المد، أغلب الفروقات بين الألف والهمزات، والمسافات الزائدة.

## الطباعة

من صفحة البحث أو لوحة الإدارة اضغط "طباعة النتيجة". سيظهر تقرير منسق بحجم A4 يحتوي على اسم المؤسسة، بيانات الطالب، جدول العلامات، المجموع، النسبة، التقدير، تاريخ الإصدار، وتوقيع الإدارة.

## تعديل اسم المؤسسة والشعار

من لوحة الإدارة يمكن تعديل:

- اسم المؤسسة
- عنوان التقرير
- رابط أو مسار الشعار
- توقيع الإدارة
- كلمة مرور الإدارة

يمكن أيضًا تعديل القيم الافتراضية من بداية ملف `data.js` داخل كائن `settings`.

إذا كان لديك شعار داخل المشروع، أنشئ مجلدًا باسم `assets` وضع الصورة داخله مثل `assets/logo.png`، ثم اكتب هذا المسار في خانة الشعار أو داخل `logoUrl`.
